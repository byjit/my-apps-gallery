# Overview

A React tanstack router boilerplate with vite and tailwindcss.

## scripts

- `pnpm install` - to install dependencies
- `pnpm run dev` - to run in development mode
- `pnpm run build` - to run in production mode
- `pnpm test` - to run tests
- `pnpm run format` - to format the code
- `pnpm run lint` - to lint the code
- `pnpm run check` - to check the code

## Tech stack
- `Vitest` for testing
- `Shadcn` for UI components
- `Tailwind CSS` for styling
- `Biome` for linting and formatting
- `TanStack Router` for routing
- `TanStack Query` for data fetching
- `T3Env` for environment variables
- `zustand` for state management
- `Vite` for bundling




## NOTE

- You can use T3Env to add type safety to your environment variables. Add Environment variables to the `src/env.mjs` file.
- The tanstack router setup is a file based router. Which means that the routes are managed as files in `src/routes`. 
- You don't need to handle the contents of `src/routeTree.gen.ts`, they are automatically generated by tanstack router.
- To use SPA (Single Page Application) navigation, use : 
  ```tsx
  import { Link } from "@tanstack/react-router";
  <Link to="/about">About</Link>
  ```
- the layout is located in `src/routes/__root.tsx`
